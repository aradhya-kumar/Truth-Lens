

## TruthLens — Deepfake Detection Website

A dark, techy landing page with a built-in media analysis tool for deepfake detection.

### Design Direction
- **Dark theme** with deep navy/charcoal backgrounds
- **Neon cyan/green accents** for highlights, borders, and interactive elements
- **Grid patterns, scan lines, and subtle glow effects** for a cyber/AI aesthetic
- **Monospace accents** mixed with clean sans-serif typography

### Page Structure

#### 1. Hero Section
- Bold headline: "See Through the Lies" or similar
- Animated scanning/grid background effect
- Glowing CTA button to scroll to the detection tool
- Brief tagline about AI-powered deepfake detection

#### 2. How It Works (3-step process)
- Upload → Analyze → Results
- Icon cards with neon borders and glow effects
- Clean, visual explanation of the detection pipeline

#### 3. Features Section
- Cards highlighting key capabilities: image analysis, video analysis, confidence scoring, real-time detection
- Subtle hover animations with glow effects

#### 4. Detection Tool Section
- Upload area with drag-and-drop support (images)
- Simulated analysis animation (scanning effect over the image)
- Results display: authenticity score, confidence meter, visual indicators (green = authentic, red = likely deepfake)
- Note: This will be a UI demo — no actual AI backend, but the interface will be fully designed

#### 5. Footer
- Links, social icons, copyright

### Key Details
- Fully responsive (mobile + desktop)
- Smooth scroll navigation
- All frontend — detection tool shows a simulated/mock analysis flow
- Animated elements: pulsing glows, scan lines, progress bars
